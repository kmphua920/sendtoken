# sendtoken
Python script to automate batch token sends to multiple addresses 

## Installation

1. Check your Python version to make sure Python3 is installed
2. Download pip
3, Install pip


```bash
python3 --version
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python3 get-pip.py
```

## Usage

```bash
pip sendtoken.py
```
